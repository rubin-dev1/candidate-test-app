# candidate-test-app
Веб-приложение для онлайн-тестирования кандидатов. Позволяет проводить оценку знаний с автоматической проверкой ответов, ограничением доступа по email и сохранением результатов в Google Sheets. Подходит для HR, рекрутеров и технических интервьюеров для быстрого отбора кандидатов.
